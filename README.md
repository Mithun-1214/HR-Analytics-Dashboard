# HR Analytics Dashboard 📊

## 📌 Project Overview
Developed an interactive HR Analytics Dashboard using Power BI to analyze employee attrition, job satisfaction, salary hike impact, and workforce demographics.  

This dashboard helps HR teams make data-driven decisions by identifying key trends and patterns in employee data.

---

## 🎯 Objectives
- Analyze employee attrition trends
- Understand job satisfaction levels
- Study salary hike impact on attrition
- Examine department-wise and demographic insights

---

## 🛠 Tools & Technologies Used
- Power BI
- DAX
- Data Modeling
- Excel (Data Cleaning)

---

## 📊 Key KPIs
- Total Employees
- Attrition Count
- Attrition Rate (%)
- Average Salary
- Average Age
- Job Satisfaction Score

---

## 📈 Dashboard Features
- Interactive filters & slicers
- Department-wise attrition analysis
- Gender and age group distribution
- Salary hike impact visualization
- Job role comparison
